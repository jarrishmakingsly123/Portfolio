Workflow Automation

01
File Arrival Detection
EPS receives files from RSDS as well as manual files. Manual files arrive at a specific designated location.
02
Control-M File Check
A Control-M Connect:Direct FW job checks whether the file has landed in the expected path or not.
03
File Transfer to Unix BIH
If the file has landed, the next job transfers the file path to the Unix BIH path for processing.
04
DataStage ETL Processing
Control-M load job triggers the DataStage batches and loads data into the respective target tables.
05
Data Validation
Specific data verification queries run in BTEQ and are triggered through Control-M for quality assurance.
06
Business Impact: Reduced Error Rate
By automating previously manual steps, human interaction is minimised and error rates are significantly reduced.
